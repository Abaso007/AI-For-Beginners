# 构建本体

构建知识库的核心是对一个主题的事实进行分类建模。选择一个主题——比如一个人、一个地方或一个事物——然后为该主题构建一个模型。使用本课中描述的一些技术和建模策略。例如，可以创建一个包含家具、灯具等的客厅本体。客厅与厨房有什么不同？浴室呢？你如何判断这是一个客厅而不是餐厅？使用 [Protégé](https://protege.stanford.edu/) 来构建你的本体。
