**Artificial Intelligence** (AI) အပေါ် ၁၂ ပတ်၊ ၂၄ သင်ခန်းစာပါသော သင်ရိုးညွှန်းတမ်းကို လေ့လာပါ။ ၎င်းတွင် လက်တွေ့သင်ခန်းစာများ၊ စမ်းမေးခွန်းများနှင့် လက်တွေ့လေ့ကျင့်ခန်းများ ပါဝင်သည်။ သင်ရိုးညွှန်းတမ်းသည် အခြေခံအဆင့်အတွက် သင့်လျော်ပြီး TensorFlow နှင့် PyTorch ကိရိယာများ၊ AI အပေါ် သမိုင်းနှင့် ကျင့်ဝတ်များကိုလည်း ဖော်ပြထားသည်။

## သင်လေ့လာမည့်အရာများ  

**[သင်ခန်းစာ၏ Mindmap](http://soshnikov.com/courses/ai-for-beginners/mindmap.html)**  

ဒီသင်ရိုးညွှန်းတမ်းတွင် သင်လေ့လာမည့်အရာများမှာ -  

* Artificial Intelligence အပေါ် အမျိုးမျိုးသောနည်းလမ်းများ၊ အထူးသဖြင့် **Knowledge Representation** နှင့် reasoning ([GOFAI](https://en.wikipedia.org/wiki/Symbolic_artificial_intelligence)) အပါအဝင် "symbolic approach" ကိုလေ့လာပါ။  
* **Neural Networks** နှင့် **Deep Learning** ကိုလေ့လာပါ။ ၎င်းသည် ခေတ်သစ် AI ၏ အခြေခံဖြစ်သည်။ TensorFlow နှင့် PyTorch ကိရိယာများကို အသုံးပြု၍ အဓိကအကြောင်းအရာများကို ရှင်းလင်းပြသမည်။  
* **Neural Architectures** ကို ရုပ်ပုံများနှင့် စာသားများနှင့်အလုပ်လုပ်ရန်အတွက် အသုံးပြုပါမည်။  
* **Genetic Algorithms** နှင့် **Multi-Agent Systems** ကဲ့သို့သော နည်းလမ်းများကိုလည်း လေ့လာပါမည်။  

ဒီသင်ရိုးညွှန်းတမ်းတွင် မပါဝင်မည့်အရာများ -  

> [ဒီသင်ခန်းစာအတွက် Microsoft Learn collection တွင် အပိုအရင်းအမြစ်များကို ရှာဖွေပါ](https://learn.microsoft.com/en-us/collections/7w28iy2xrqzdj0?WT.mc_id=academic-77998-bethanycheum)  

* **AI in Business** ကို အသုံးပြုခြင်းဆိုင်ရာ စီးပွားရေးအခန်းကဏ္ဍများ။ [AI for business users](https://docs.microsoft.com/learn/paths/introduction-ai-for-business-users/?WT.mc_id=academic-77998-bethanycheum) သင်ခန်းစာကို Microsoft Learn တွင် လေ့လာပါ။  
* **Classic Machine Learning** ကို [Machine Learning for Beginners Curriculum](http://github.com/Microsoft/ML-for-Beginners) တွင် ရှင်းလင်းထားသည်။  
* **Cognitive Services** အသုံးပြု၍ AI အလေ့အကျင့်များကို ဖန်တီးခြင်း။  
* **Cloud Frameworks** အထူးသဖြင့် [Azure Machine Learning](https://azure.microsoft.com/services/machine-learning/?WT.mc_id=academic-77998-bethanycheum) နှင့် [Microsoft Fabric](https://learn.microsoft.com/en-us/training/paths/get-started-fabric/?WT.mc_id=academic-77998-bethanycheum) စသည်တို့ကို လေ့လာပါ။  
* **Conversational AI** နှင့် **Chat Bots** ကို [Create conversational AI solutions](https://docs.microsoft.com/learn/paths/create-conversational-ai-solutions/?WT.mc_id=academic-77998-bethanycheum) သင်ခန်းစာတွင် လေ့လာပါ။  
* **Deep Mathematics** ကို [Deep Learning](https://www.amazon.com/Deep-Learning-Adaptive-Computation-Machine/dp/0262035618) စာအုပ်မှ လေ့လာပါ။  

AI in the Cloud အပေါ် အနည်းငယ်သော အကျဉ်းချုပ်ကို [Get started with artificial intelligence on Azure](https://docs.microsoft.com/learn/paths/get-started-with-artificial-intelligence-on-azure/?WT.mc_id=academic-77998-bethanycheum) သင်ခန်းစာမှ စတင်ပါ။

## သင်ခန်းစာတိုင်းတွင် ပါဝင်သည်

* ကြိုတင်ဖတ်ရှုရန်အကြောင်းအရာများ  
* **PyTorch** သို့မဟုတ် **TensorFlow** framework အတွက် သီးသန့်ရေးသားထားသော Jupyter Notebooks များ။ ဤ notebooks များတွင် သီအိုရီဆိုင်ရာအကြောင်းအရာများလည်း ပါဝင်ပြီး သင်ခန်းစာကို နားလည်ရန်အတွက် PyTorch သို့မဟုတ် TensorFlow ၏ အနည်းဆုံးတစ်မျိုးကို လေ့လာရန်လိုအပ်ပါသည်။  
* **Labs** သင်ခန်းစာအချို့တွင် ရရှိနိုင်ပြီး သင်လေ့လာထားသောအကြောင်းအရာများကို သတ်မှတ်ပြဿနာတစ်ခုတွင် လက်တွေ့ကျင့်သုံးနိုင်စေရန် အခွင့်အရေးပေးပါသည်။  
* အချို့သောအပိုင်းများတွင် [**MS Learn**](https://learn.microsoft.com/en-us/collections/7w28iy2xrqzdj0?WT.mc_id=academic-77998-bethanycheum) ၏ ဆက်စပ်အကြောင်းအရာများကို လင့်ခ်များဖြင့် ပံ့ပိုးထားပါသည်။

## စစ်ဆေးမေးခွန်းများ
> **မေးခွန်းများအကြောင်းမှတ်ချက်**: မေးခွန်းများအားလုံးသည် Quiz-app ဖိုလ်ဒါတွင် etc\quiz-app တွင်ပါရှိပြီး၊ [အွန်လိုင်းတွင် ဒီမှာ](https://ff-quizzes.netlify.app/) တွေ့နိုင်ပါသည်။ မေးခွန်းများကို သင်ခန်းစာများအတွင်းမှ ချိတ်ဆက်ထားပြီး၊ quiz app ကို ဒေသတွင်းတွင် အလုပ်လည်စေခြင်း သို့မဟုတ် Azure တွင် တင်သွင်းနိုင်ပါသည်။ `quiz-app` ဖိုလ်ဒါအတွင်းရှိ လမ်းညွှန်ချက်များကို လိုက်နာပါ။ မေးခွန်းများကို တဖြည်းဖြည်း ဒေသိယပြုလုပ်နေပါသည်။
