# အခြေခံအဆင့်အတုရုပ်သံနည်းပညာ - သင်ရိုးညွှန်းတမ်း

**အတုရုပ်သံနည်းပညာ** (AI) ၏ကမ္ဘာကို ၁၂ ပတ်၊ ၂၄ သင်ခန်းစာများပါဝင်သော သင်ရိုးညွှန်းတမ်းဖြင့် လေ့လာပါ။ ၎င်းတွင် လက်တွေ့ကျသော သင်ခန်းစာများ၊ စမ်းမေးခွန်းများနှင့် လက်တွေ့လုပ်ငန်းများပါဝင်သည်။ သင်ရိုးညွှန်းတမ်းသည် အခြေခံအဆင့်အတွက် သင့်တော်ပြီး TensorFlow နှင့် PyTorch ကဲ့သို့သော ကိရိယာများနှင့် AI ရဲ့ စည်းကမ်းများကိုလည်း ဖော်ပြထားသည်။

## သင်လေ့လာမည့်အရာများ  

**[သင်ရိုးညွှန်းတမ်း၏ Mindmap](http://soshnikov.com/courses/ai-for-beginners/mindmap.html)**  

ဒီသင်ရိုးညွှန်းတမ်းတွင် သင်လေ့လာမည့်အရာများမှာ -  

* အတုရုပ်သံနည်းပညာ၏ အမျိုးမျိုးသောနည်းလမ်းများ၊ **Knowledge Representation** နှင့် reasoning ([GOFAI](https://en.wikipedia.org/wiki/Symbolic_artificial_intelligence)) ပါဝင်သော "ရိုးရာ" သင်္ချာနည်းလမ်းများကိုပါဝင်သည်။  
* **Neural Networks** နှင့် **Deep Learning**၊ ယခုခေတ် AI ၏ အခြေခံဖြစ်သည်။ ဒီအရေးကြီးသောအကြောင်းအရာများကို TensorFlow နှင့် PyTorch ကဲ့သို့သော framework နှစ်ခုတွင် code အသုံးပြု၍ ရှင်းလင်းပြသမည်။  
* **Neural Architectures** ကို ပုံများနှင့် စာသားများနှင့်အလုပ်လုပ်ရန်အတွက် အသုံးပြုမည်။ နောက်ဆုံးပေါ်နည်းလမ်းများကို ဖော်ပြမည်ဖြစ်သော်လည်း အချို့နည်းလမ်းများမှာ နောက်ဆုံးပေါ်နည်းလမ်းများမပါဝင်နိုင်ပါ။  
* **Genetic Algorithms** နှင့် **Multi-Agent Systems** ကဲ့သို့သော နည်းလမ်းများကိုလည်း လေ့လာမည်။  

ဒီသင်ရိုးညွှန်းတမ်းတွင် မပါဝင်မည့်အရာများ -  

> [ဒီသင်ရိုးညွှန်းတမ်းအတွက် Microsoft Learn collection တွင် အပိုဆောင်းအရင်းအမြစ်များကို ရှာဖွေပါ](https://learn.microsoft.com/en-us/collections/7w28iy2xrqzdj0?WT.mc_id=academic-77998-bethanycheum)  

* **AI in Business** ကို အသုံးပြုခြင်းဆိုင်ရာ စီးပွားရေးအခွင့်အလမ်းများ။ [Introduction to AI for business users](https://docs.microsoft.com/learn/paths/introduction-ai-for-business-users/?WT.mc_id=academic-77998-bethanycheum) သင်ခန်းစာလမ်းကြောင်းကို Microsoft Learn တွင် လေ့လာပါ၊ သို့မဟုတ် [AI Business School](https://www.microsoft.com/ai/ai-business-school/?WT.mc_id=academic-77998-bethanycheum) ကို [INSEAD](https://www.insead.edu/) နှင့် ပူးပေါင်းဖွံ့ဖြိုးထားသည်။  
* **Classic Machine Learning** ကို [Machine Learning for Beginners Curriculum](http://github.com/Microsoft/ML-for-Beginners) တွင် ရှင်းလင်းဖော်ပြထားသည်။  
* **[Cognitive Services](https://azure.microsoft.com/services/cognitive-services/?WT.mc_id=academic-77998-bethanycheum)** ကို အသုံးပြု၍ တည်ဆောက်ထားသော လက်တွေ့ AI အက်ပလီကေးရှင်းများ။ [vision](https://docs.microsoft.com/learn/paths/create-computer-vision-solutions-azure-cognitive-services/?WT.mc_id=academic-77998-bethanycheum), [natural language processing](https://docs.microsoft.com/learn/paths/explore-natural-language-processing/?WT.mc_id=academic-77998-bethanycheum), **[Generative AI with Azure OpenAI Service](https://learn.microsoft.com/en-us/training/paths/develop-ai-solutions-azure-openai/?WT.mc_id=academic-77998-bethanycheum)** နှင့် အခြား module များကို Microsoft Learn တွင် စတင်ပါ။  
* **Cloud Frameworks** များ၊ [Azure Machine Learning](https://azure.microsoft.com/services/machine-learning/?WT.mc_id=academic-77998-bethanycheum), [Microsoft Fabric](https://learn.microsoft.com/en-us/training/paths/get-started-fabric/?WT.mc_id=academic-77998-bethanycheum), သို့မဟုတ် [Azure Databricks](https://docs.microsoft.com/learn/paths/data-engineer-azure-databricks?WT.mc_id=academic-77998-bethanycheum) ကဲ့သို့သော ML Framework များ။ [Build and operate machine learning solutions with Azure Machine Learning](https://docs.microsoft.com/learn/paths/build-ai-solutions-with-azure-ml-service/?WT.mc_id=academic-77998-bethanycheum) နှင့် [Build and Operate Machine Learning Solutions with Azure Databricks](https://docs.microsoft.com/learn/paths/build-operate-machine-learning-solutions-azure-databricks/?WT.mc_id=academic-77998-bethanycheum) သင်ခန်းစာလမ်းကြောင်းများကို စတင်ပါ။  
* **Conversational AI** နှင့် **Chat Bots**။ [Create conversational AI solutions](https://docs.microsoft.com/learn/paths/create-conversational-ai-solutions/?WT.mc_id=academic-77998-bethanycheum) သင်ခန်းစာလမ်းကြောင်းကို သီးသန့်လေ့လာပါ၊ သို့မဟုတ် [ဒီ blog post](https://soshnikov.com/azure/hello-bot-conversational-ai-on-microsoft-platform/) ကို အသေးစိတ်ဖတ်ရှုပါ။  
* **Deep Mathematics** ကို deep learning အတွက် လေ့လာရန်။ [Deep Learning](https://www.amazon.com/Deep-Learning-Adaptive-Computation-Machine/dp/0262035618) ကို Ian Goodfellow, Yoshua Bengio နှင့် Aaron Courville မှ ရေးသားထားပြီး [https://www.deeplearningbook.org/](https://www.deeplearningbook.org/) တွင်လည်း ရရှိနိုင်သည်။  

_Cloud_ အတုရုပ်သံနည်းပညာဆိုင်ရာ gentle introduction အတွက် [Get started with artificial intelligence on Azure](https://docs.microsoft.com/learn/paths/get-started-with-artificial-intelligence-on-azure/?WT.mc_id=academic-77998-bethanycheum) သင်ခန်းစာလမ်းကြောင်းကို စတင်ပါ။

## သင်ခန်းစာတစ်ခုစီတွင် ပါဝင်သည်

* ကြိုတင်ဖတ်ရန်အကြောင်းအရာများ
* Jupyter Notebooks များကို အကောင်အထည်ဖော်နိုင်သည်။ Framework (**PyTorch** သို့မဟုတ် **TensorFlow**) အပေါ်မူတည်၍ အထူးသီးသန့်ဖြစ်သည်။ Notebook တွင် သီအိုရီအကြောင်းအရာများစွာပါဝင်ပြီး၊ အကြောင်းအရာကို နားလည်ရန် PyTorch သို့မဟုတ် TensorFlow Notebook တစ်ခုခုကို အနည်းဆုံး လေ့လာရမည်။
* **Labs** သင်ခန်းစာအချို့တွင် ရရှိနိုင်ပြီး၊ သင်လေ့လာထားသောအကြောင်းအရာများကို အထူးပြဿနာတစ်ခုတွင် အသုံးချရန် အခွင့်အရေးပေးသည်။
* အချို့သောအပိုင်းများတွင် [**MS Learn**](https://learn.microsoft.com/en-us/collections/7w28iy2xrqzdj0?WT.mc_id=academic-77998-bethanycheum) မှ ဆက်စပ်အကြောင်းအရာများကို လင့်ခ်များဖြင့် ပံ့ပိုးထားသည်။

## စတင်ရန်

### 🎯 AI အသစ်စတင်လေ့လာသူများအတွက်! ဒီနေရာမှ စတင်ပါ!

AI ကို လုံးဝအသစ်စတင်လေ့လာသူများအတွက် လက်တွေ့လုပ်ဆောင်နိုင်သော ဥပမာများကို [**Beginner-Friendly Examples**](./examples/README.md) တွင် ကြည့်ရှုပါ! ဤအကြောင်းအရာများတွင် ပါဝင်သည်-

- 🌟 **Hello AI World** - သင်၏ပထမဆုံး AI အစီအစဉ် (pattern recognition)
- 🧠 **Simple Neural Network** - Neural Network ကို အခြေခံကနေ တည်ဆောက်ခြင်း

- 💬 **Text Sentiment** - အပေါင်း/အပြား စာသားကို ခွဲခြားခြင်း

ဤဥပမာများကို AI အကြောင်းအရာများကို နားလည်ရန် အကူအညီပေးရန် ရည်ရွယ်ထားပြီး၊ အပြည့်အစုံသင်ရိုးကို စတင်မတိုင်မီ လေ့လာနိုင်ပါသည်။
