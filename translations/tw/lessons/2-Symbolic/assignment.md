# 建立本體

建立知識庫的核心在於對某個主題進行事實模型的分類。選擇一個主題，例如人物、地點或事物，然後建立該主題的模型。使用本課中描述的一些技術和模型構建策略。一個例子可能是建立一個客廳的本體，包括家具、燈光等等。客廳與廚房有什麼不同？浴室呢？你如何判斷它是客廳而不是餐廳？使用 [Protégé](https://protege.stanford.edu/) 來建立你的本體。
