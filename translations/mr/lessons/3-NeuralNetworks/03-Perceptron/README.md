## рдкрд░рд╕реЗрдкреНрдЯреНрд░реЙрдирдЪреЗ рдкреНрд░рд╢рд┐рдХреНрд╖рдг

рдЬрд┐рдереЗ:

* рдпреЛрдЧ рддреНрдпрд╛ рдкреНрд░рд╢рд┐рдХреНрд╖рдг рдбреЗрдЯрд╛ рдкреЙрдЗрдВрдЯреНрд╕ i рд╡рд░ рдШреЗрддрд▓рд╛ рдЬрд╛рддреЛ рдЬреНрдпрд╛рдореБрд│реЗ рдЪреБрдХреАрдЪреЗ рд╡рд░реНрдЧреАрдХрд░рдг рд╣реЛрддреЗ

рдЗрдереЗ ╬╖ рдореНрд╣рдгрдЬреЗ **рд▓рд░реНрдирд┐рдВрдЧ рд░реЗрдЯ**, рдЖрдгрд┐ тИЗE(w) рдореНрд╣рдгрдЬреЗ E рдЪрд╛ **рдЧреНрд░реЗрдбрд┐рдпрдВрдЯ**. рдЧреНрд░реЗрдбрд┐рдпрдВрдЯрдЪреА рдЧрдгрдирд╛ рдХреЗрд▓реНрдпрд╛рдирдВрддрд░, рдЖрдкрд▓реНрдпрд╛рд▓рд╛ рдорд┐рд│рддреЗ:

w<sup>(t+1)</sup> = w<sup>(t)</sup> + тИС╬╖x<sub>i</sub>t<sub>i</sub>

рдкрд╛рдпрдердирдордзреАрд▓ рдЕрд▓реНрдЧреЛрд░рд┐рджрдо рдЕрд╕реЗ рджрд┐рд╕рддреЗ:

```python
def train(positive_examples, negative_examples, num_iterations = 100, eta = 1):

    weights = [0,0,0] # Initialize weights (almost randomly :)
        
    for i in range(num_iterations):
        pos = random.choice(positive_examples)
        neg = random.choice(negative_examples)

        z = np.dot(pos, weights) # compute perceptron output
        if z < 0: # positive example classified as negative
            weights = weights + eta*weights.shape

        z  = np.dot(neg, weights)
        if z >= 0: # negative example classified as positive
            weights = weights - eta*weights.shape

    return weights
```
